### Build Database:
#### Step 1
Navigate to: ```Visual Studio > Tools > NuGet Pckg Manager > Console```
#### Step 2
Type: ```enable-migrations```
#### Step 3
Type: ```add-migration MyMigration```
#### Step 4
Type: ```update-database```
